unsafeAt :: Text -> Attribute a Source #

Access the value of a node's attribute in text form (equivalent to XPath's @). Unsafe because it has no way to check if the attribute name provided matches the schema. It is recommend to call this function once only for each attribute in your schema. The resulting value can be re-used.
